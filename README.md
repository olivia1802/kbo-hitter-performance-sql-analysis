# KBO Hitter Performance Analysis Using SQL

This project analyzes offensive performance in the Korean Baseball Organization (KBO) League using SQL and batting statistics from the 2021–2025 seasons. The analysis focuses on offensive production and advanced metrics such as OPS, wRC+, and WAR.

## Project Goals

The objective of this project was to identify elite offensive seasons in the KBO League and explore how advanced baseball metrics can be used to evaluate player and team performance beyond traditional statistics.

## Research Questions

* Which individual player seasons recorded the highest OPS values between 2021 and 2025?
* Which player seasons generated the most overall value based on WAR?
* Which player seasons generated the strongest offensive production based on wRC+?
* Which player seasons generated high offensive production despite relatively low home run totals?
* Which teams demonstrated the strongest offensive production based on average wRC+?

## Key Findings

* Na Sung-bum's 2023 season and Kim Do-young's 2024 season ranked among the most productive offensive seasons in the dataset.
* Lee Jung-hoo's 2022 season generated the highest WAR between 2021 and 2025.
* Na Sung-bum's 2023 season recorded the highest wRC+ in the dataset, indicating exceptional offensive production relative to league average.
* Players such as Lee Jung-hoo and Hong Chang-ki generated elite offensive value despite hitting relatively few home runs.
* LG recorded the highest average team wRC+ among qualified KBO hitters during the 2021–2025 period.
